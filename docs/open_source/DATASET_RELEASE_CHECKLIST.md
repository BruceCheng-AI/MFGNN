# Dataset Release Checklist

- [ ] Confirm that every raw source can be redistributed publicly.
- [ ] Choose and add a dataset license.
- [ ] Decide whether to publish one combined dataset package or a core processed package plus a raw supplementary package.
- [ ] Include `StreetSZ/` and the processed tables needed to reproduce the modeling pipeline.
- [ ] Add `DATASET_CARD.md` and `DATASET_DICTIONARY.md`.
- [ ] Add version information and a release date.
- [ ] Add file checksums for the final uploaded package.
- [ ] Verify that no personal data, institution-restricted data, or third-party proprietary content remains.
- [ ] Document any preprocessing or imputation steps applied before release.
- [ ] State the intended task, target variables, temporal resolution, and spatial unit clearly.
- [ ] Provide one canonical download link to use from the code repository README.
